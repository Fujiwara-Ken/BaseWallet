# SampleSolanaWallet

solana-suite を用いたウォレットシステム

<https://github.com/atonoy/solana-suite>

## 1. `git clone`

```shell
git clone git@github.com:Fujiwara-Ken/BaseWallet.git
```

## 2. `yarn install`

`yarn install`

```shell
# v1.22.15であることを確認
yarn -v

#パッケージをインストール
yarn install
```

## 3. `yarn dev`

`yarn dev`でローカルサーバーを立ち上げ。

```shell
yarn dev
```
